# 📘 AlgoNotes – Algorithm & Data Structure Lecture Notes

AlgoNotes is a static website that provides concise lecture notes and summaries for **Programming Algorithms** and **Data Structures**. It is built with pure HTML, CSS, and JavaScript – no frameworks or backends required. The design uses a soft blue-gray color palette to ensure a clean, readable, and modern learning experience.

## 🎯 Purpose

This project serves as a personal or shared knowledge base for students and developers who want to quickly review fundamental algorithm and data structure concepts. It includes:
- Step‑by‑step explanations
- Code snippets
- Big‑O complexity comparisons
- Practical learning tips

## ✨ Features

- **Fully static** – fast, secure, and easy to host (GitHub Pages, Netlify, Cloudflare Pages, etc.)
- **Responsive layout** – works seamlessly on desktabs, tablets, and mobile devices
- **Clean information architecture** – separate sections for Algorithms, Data Structures, and a detailed Summary / Cheatsheet
- **Interactive cards** – each topic displays a short description and a ready‑to‑use code example
- **Statistics counter** – automatically shows the total number of topics and code examples
- **Smooth navigation** – sticky header with active section highlighting
- **Mobile friendly** – hamburger menu for small screens
- **Custom color palette** – soft blue tones that reduce eye strain and improve readability

## 📚 Content Overview

### 1. Algorithms
- Bubble Sort
- Binary Search
- Recursion (Factorial)

### 2. Data Structures
- Stack (LIFO)
- Queue (FIFO)
- Linked List
- Binary Tree

### 3. Summary & Cheatsheet (extended version)
- Big‑O complexity table (average & worst case)
- Detailed explanations of arrays, stacks, queues, linked lists, hash tables, and BST
- Additional sorting algorithms (Quick Sort, Merge Sort)
- Practical code snippet (Quick Sort implementation)
- Effective study tips and resource recommendations

All code snippets are displayed in a dark, monospace block for easy reading.
